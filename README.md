# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- リモートリポジトリ：ネット上に配置して複数人で共有するためのリポジトリ
- ローカルリポジトリ：自分のPC上に配置するためのリポジトリ

## プッシュとマージの違いは何でしょうか？

- プッシュ：ローカルでの変更をリモートへ送信する
- マージ：ブランチ同士の変更履歴を取り込む

## コミットとプッシュの違い

- コミット：ローカルに変更履歴を残す
- プッシュ：ローカルでの変更をリモートへ送信する

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

- 変更内容をわかりやすいように記載する

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

- ローカルでマージするフロー：レビューなしでローカルにてブランチをマージし、リモートへプッシュする
- プルリクエストでマージするフロー：ローカルで内容を作成後、リモートへプッシュし、レビューアにてコードレビュー後、マージする

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- 先にマージされた変更内容を取り込む
- 後にマージしようとしている変更内容を取り込む
- どちらの変更内容も取り込む
